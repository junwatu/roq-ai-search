## `<UserAccountDropdown/>`

### Description

The `<UserAccountDropdown/>` component can be used to show the user's avatar and their account dropdown.
It works with ROQ Authentication, and without.

</Tab></Tabs>

### Integration with ROQ Authentication

To show the UserAccountDropdown, add this snippet to your page:

<Tabs items={['Reactjs', 'Next.js']}><Tab>

```tsx
import {UserAccountDropdown} from "@roq/ui-react";

function App() {
    return <UserAccountDropdown/>;
}
```

</Tab><Tab>

```tsx
import {UserAccountDropdown} from "@roq/nextjs";

function App() {
    return <UserAccountDropdown/>;
}
```

</Tab></Tabs>

### Integration without ROQ Authentication

<Tabs items={['Reactjs', 'Next.js']}><Tab>

```tsx
import {UserAccountDropdown} from "@roq/ui-react";

function App() {
    const handleSignOut = () => {
        //your sign-out logic
    };

    return <UserAccountDropdown onSignOut={handleSignOut}/>;
}
```

</Tab><Tab>

```tsx
import {UserAccountDropdown} from "@roq/nextjs";

    return <UserAccountDropdown onSignOut={handleSignOut}/>;
```

## `<UserProfile/>`

### Description

The `<UserProfile/>` component can be used to show the user's profile information, along with the ability to edit the profile.
It includes an avatar uploader, and a form to edit the user's password, and other profile information.
