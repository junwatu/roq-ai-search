# How to Create Custom Translations

The translation management can be done in the **Translations** section of the ROQ console. On this page, you can see all the translation keys list, the default translation, translation values, and the status of the translated keys, whether being translated or not. 

<Callout type="info">
The default translation for the generated application is in `English, United States (en-US)` language.
</Callout>

![Generated application translation default](/app-default-local-translation.png)

Translations customization can be done by editing or updating the value of the translation, and translation keys and furthermore by adding, editing, or updating translation locales.

## Edit Translation Values

To change or edit the translation values, please click the **Translate Key** button (pen icon).

<Callout type="info">
The default selected translation locale in the **Translations** list is `English, United States (en-US)`, for other locales, choose in the **Select Locale** dropdown.
</Callout>

![translation editing](/translates-editing.png)
