# Customization

Customize the managed login and registration pages

## Configure Texts and Links

All texts and links can be adapted. Login to 
[ROQ Console](https://console.roq.tech/settings/authentication-form/theming) and go to `Authentication` ➔ `Configuration`

![Configuration](/new-auth-configuration.png)


## Theming

<Callout type="info" emoji="ℹ️">
The customization supports two modes: Light Mode and Dark Mode. Each mode has different settings and values. To apply changes to both style modes, you need to set values for each mode.
</Callout>

You can change the look and feel of your login and registration page in 
[ROQ Console](https://console.roq.tech/settings/authentication-form/theming) and go to `Authentication` ➔ `Theming`

![Theming](/new-auth-theming.png)

## Logo and Favicon

The Registration or Login logo can be customized by changing the `Logo` field with the URL of your logo.

![Logo URL setting](/logo-url.png)

Don't forget to save the settings and check the preview using the preview button

![Login and registration form preview](/login-registration-preview.png)

The favicon is easy to customize by changing the default `Favicon`

![Favicon setting](/favicon-custom-upload.png)

Please note that the customization support two style modes: Light and Dark Mode
