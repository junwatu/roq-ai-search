# How to Add Social Sign-in

Before add social sign-ins to registration or login form, make sure to activate and configure them, check out the docs [here](/authentication/social-sign-ins).

For example to add Google Sign-in to login form, just enable it and don't forget to click `Save` button to save the setting.
