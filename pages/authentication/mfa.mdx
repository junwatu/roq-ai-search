# Multi-Factor Authentication

ROQ provide additional layer of protection which is **Multi-Factor Authentication** or simply called **MFA**. With this security enhancement you can use your phone number as additional application authentication.

To enable the Multi-factor Authentication please go to the ROQ Console.
