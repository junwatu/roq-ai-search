# Tutorials

## Notify User Groups Using Notify API

ROQ offers an API that allows users or user groups to receive notifications. With this feature, you can easily send important information to multiple or specific users or groups. For more information on how to send notifications, please refer to this [tutorial](/user-groups/tutorials/notify-user-groups).

## Send Email to User Groups Using Mails API

In various scenarios, there arises a requirement to send emails to particular sets of individuals or groups. To achieve this, we can employ the User Groups API or GraphQL and Mails API. A useful guide on how to send emails to groups can be found in this [tutorial](/user-groups/tutorials/send-email-to-groups).
