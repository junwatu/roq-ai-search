# Edit Data Model

ROQ provides a database model or schema editor. It is designed so you can easily customize the generated application data model to meet your business needs. 

You can access the Data Model Editor feature in ROQ Console. Go to **Home** â†’ **Get Started** and find the **Step 3: Fine-tune your database model** section and click the **Edit Data Model** button.

![Edit schema button](/edit-schema-link.png)

## Data Model Editor

With this Data Model Editor, you can: **Add New Model**, **Add Fields**, **Create and Edit Relationships**.
