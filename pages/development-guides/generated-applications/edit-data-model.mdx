# Edit Data Model

ROQ provides a database model or schema editor. You can access this feature in ROQ Console. Go to **Home** â†’ **Get Started** and find the **Step 3: Fine-tune your database model** section and then click the **Edit Data Model** button.

## Data Model Editor

ROQ Data Model Editor is designed so you can easily customize the generated application data model to meet your business needs. With this data model editor you can: **Add New Model**, **Add Fields**, **Create and Edit Relationships**.
